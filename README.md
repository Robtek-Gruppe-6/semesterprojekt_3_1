## Setup
To run the code you need to have the following packages
sounddevice
pyaudio
numpy 
scipy
matplotlib
curses (If linux)
windows-curses (if windows)

### For Linux
```
pip install numpy
pip install sounddevice
pip install scipy
pip install pyaudio
pip install matplotlib
pip install curses
```
### For Windows
```
pip install numpy
pip install sounddevice
pip install scipy
pip install pyaudio
pip install matplotlib
pip install windows-curses
```
